== Description ==

ALD Login Page can manage login page flexibly with simple markup with the help of wordpress. Just another Login page customization plugin. Simple but flexible.

== Installation ==

1. Upload the entire `ald-login-page` folder to the `/wp-content/plugins/` directory.
1. Activate the plugin through the 'Plugins' menu in WordPress.

You will find 'ALD Login Page' menu in your WordPress admin panel Setting.
